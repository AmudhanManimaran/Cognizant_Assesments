# React Hands-on: Creating and Rendering Multiple Components

## Objective

The objective of this hands-on exercise is to understand the fundamentals of React components by creating and rendering multiple class components in a React application.

The application demonstrates how independent components can be created, exported, imported, and rendered together within a parent component.

---

## Prerequisites

Before starting the exercise, ensure the following software is installed:

- Node.js (LTS version recommended)
- npm (installed with Node.js)
- Visual Studio Code

---

## Technologies Used

- React
- JavaScript (ES6)
- Node.js
- npm
- Create React App

## Steps Performed

### 1. Create the React Application

```bash
npx create-react-app studentapp
```

### 2. Navigate to the Project Directory

```bash
cd studentapp
```

### 3. Open the Project in Visual Studio Code

```bash
code .
```

### 4. Create a Components Folder

Inside the `src` directory, create a folder named:

```text
Components
```

### 5. Create the Components

Create the following files inside the `Components` folder:

- Home.js
- About.js
- Contact.js

Each file contains a React class component displaying the required message.

### 6. Update App.js

Import and render all three components inside the root component.

### 7. Run the Application

```bash
cd studentapp
npm install
npm start
```
