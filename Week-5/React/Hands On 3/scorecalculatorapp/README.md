# React Hands-on: Student Score Calculator Using Functional Components

## Objective

The objective of this hands-on exercise is to understand the fundamentals of React functional components by creating a reusable component that accepts data through **props**, applies **CSS styling**, and calculates a student's score dynamically.

The application demonstrates component reusability, props, JSX, helper functions, and external stylesheet integration in React. The lab specifically asks for a React application named `scorecalculatorapp` with a functional component named `CalculateScore` that accepts student details and calculates the average score. :contentReference[oaicite:0]{index=0}

## Steps Performed

### 1. Create the React Application

```bash
npx create-react-app scorecalculatorapp
```

### 2. Navigate to the Project Directory

```bash
cd scorecalculatorapp
```

### 3. Open the Project in Visual Studio Code

```bash
code .
```

### 4. Create the Components Folder

Inside the `src` directory, create a folder named:

```text
Components
```

Create a new file:

```text
CalculateScore.js
```

### 5. Create the Stylesheets Folder

Inside the `src` directory, create another folder named:

```text
Stylesheets
```

Create the stylesheet:

```text
mystyle.css
```

### 6. Implement the Functional Component

Create the `CalculateScore` functional component that:

- Accepts `Name`, `School`, `total`, and `goal` as props
- Calculates the score using helper functions
- Displays the student details
- Applies external CSS styling

### 7. Update App.js

Import the `CalculateScore` component and render it by passing the required props.

### 8. Run the Application

```bash
cd scorecalculatorapp
npm install
npm start
```
