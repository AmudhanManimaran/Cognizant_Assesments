Exercise 7: Financial Forecasting

1. Understanding Recursion
Recursion is a technique where a method calls itself to solve a smaller part of the same problem. It makes some problems easier to understand and implement.

2. Time Complexity
Time Complexity: O(n), where n is the number of years.
Space Complexity: O(n), because each recursive call is stored in the call stack.

3. Optimization
For very large inputs, recursion may use too much memory. The solution can be optimized by:
- Using memoization to store previously calculated results.
- Using an iterative approach (for loop), which reduces space complexity to O(1).
